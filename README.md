# CodSoft Task 2 – Landing Page

This repository contains my submission for CodSoft Task 2. It is a simple and responsive landing page built using HTML and CSS.

## 📌 Project Overview
The landing page is designed to present content in a clean and attractive way. It demonstrates my understanding of layout design, styling, and basic frontend development.

## 📁 Project Structure
- index.html   : Main HTML file (structure of the landing page)
- style.css    : CSS file (design and styling)
- README.md    : Project documentation

## ✨ Features
- Clean and modern UI design
- Well-structured layout
- Responsive design (basic)
- Styled sections (header, content, footer)
- Easy navigation

## 🛠️ Technologies Used
- HTML5
- CSS3

## 🚀 How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Open index.html in any web browser
